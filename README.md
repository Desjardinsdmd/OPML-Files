# OPML Feed Collection

A growing collection of OPML files containing curated RSS feed lists.

This repo is being built as a central place to store and organize feed collections by topic, source type, or use case. Right now, this is just getting started and currently includes one OPML file, with more to come over time.

## What’s in this repo

- OPML files with grouped RSS feeds
- Feed collections that can be imported into RSS readers and aggregators
- A starting point for building larger categorized feed libraries

## Current status

This project is in the early stages. At the moment, there is only one file in the repo, but the goal is to keep expanding it with additional OPML collections.

## How to use

1. Download the OPML file you want
2. Import it into your RSS reader of choice
3. Browse, edit, or split the feeds as needed

## Why this exists

Finding and organizing quality RSS feeds takes time. The idea behind this repo is to make that process easier by keeping feed lists in a reusable, portable format.

## Plans

- Add more OPML files
- Organize collections by niche or industry
- Clean up naming and structure as the library grows

## Contributing

This repo is just getting started, but contributions, suggestions, and feed ideas may be added later as it grows.
